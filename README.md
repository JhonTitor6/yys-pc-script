## 📖 项目简介

这是一个基于事件驱动架构的阴阳师PC端自动化挂机脚本，支持后台运行（不支持最小化），无需前台激活游戏窗口即可完成日常任务。

## ✨ 主要特性

- **事件驱动架构** - 基于事件响应机制，提高脚本稳定性和效率
- **后台操作** - 支持Windows后台图像识别、鼠标点击和键盘输入
- **多任务支持** - 提供多种常见挂机任务的自动化执行
- **轻量设计** - 架构简洁，易于扩展和维护

## 🚀 快速开始

### 环境要求
- Python 3.11+
- Windows操作系统

### 安装步骤
```bash
# 克隆项目
git clone https://github.com/username/yys-pc-script.git
cd yys-pc-script

# 安装依赖
pip install -r requirements.txt
```


### 运行方法
```bash
python xxx.py
```
